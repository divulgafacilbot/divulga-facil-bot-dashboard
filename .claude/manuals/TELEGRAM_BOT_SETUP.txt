STEP 5: Choose Bot Username (Arts Bot)

5.1 - BotFather will ask:
      "Good. Now let's choose a username for your bot.
      It must end in 'bot'. Like this, for example:
      TetrisBot or tetris_bot."

5.2 - Think of a username (this is PERMANENT, choose wisely)
      Rules:
      - MUST end with "bot" (lowercase)
      - Can contain letters, numbers, underscores
      - Must be unique (not taken by another bot)
      - Minimum 5 characters

      Examples:
      - postingarts_bot
      - mystorearts_bot
      - productsimages_bot

5.3 - Type your chosen username
      Example: postingarts_bot

5.4 - Press Enter/Send

5.5 - Wait for BotFather's response...

EXPECTED RESULT (if successful):
      "Done! Congratulations on your new bot.
      You will find it at t.me/[your_bot_username]

      profile picture for your bot, see /help for a list
      of commands.

      1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890

      Please try another one."

      → If failed: Go back to step 5.2 and try different username

- Username taken: add numbers (postingarts2024_bot)
- Invalid format: ensure it ends with "bot"
- Too short: make it at least 5 characters

STEP 6: SAVE ARTS BOT TOKEN (CRITICAL!)

6.1 - In BotFather's success message, locate the token
      It looks like this:
      1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890

      Format: [numbers]:[letters and numbers]

6.2 - SELECT THE ENTIRE TOKEN carefully
      - On phone: tap and hold, drag to select all
      - On desktop: triple-click to select line
      - On web: click and drag to select

      Make sure you select from first digit to last digit!

6.3 - COPY the token
      - On phone: tap "Copy"
      - On desktop: Ctrl+C (Windows/Linux) or Cmd+C (Mac)
      - On web: Right-click → Copy

6.4 - IMMEDIATELY paste it somewhere safe:
      - Open a text file called "telegram-tokens.txt"
      - Create a new line: "ARTS BOT TOKEN:"
      - Paste the token below it
      - Save the file

6.5 - VERIFY the token is complete:
      - Should be approximately 46 characters
      - Should contain a colon (:) in the middle
      - Should start with numbers
      - Should have letters and numbers after colon

CRITICAL: DO NOT share this token with anyone!
         It gives FULL CONTROL of your bot!

EXPECTED RESULT: Arts Bot token saved in text file

TROUBLESHOOTING:
- If token not visible: scroll up in BotFather chat
- If token lost: type /token in BotFather, select your bot

TROUBLESHOOTING COMMON ISSUES
═══════════════════════════════════════════════════════════════

Issue: Bot doesn't respond to messages
Solution: This is normal until you deploy your code. The bot token
         is valid, but there's no server listening to it yet.

Issue: Can't find bot by username
Solution: Double-check spelling. Username is case-sensitive.
         Must include "bot" at the end.

Issue: Token doesn't work
Solution: Make sure you copied the ENTIRE token, including the
         part after the colon. Check for extra spaces in .env.

Issue: "Username is already taken"
Solution: Try a different username. Add numbers or underscores.

Issue: BotFather doesn't respond
Solution: Wait a few seconds. BotFather sometimes has delays.
         If it persists, restart Telegram app.
